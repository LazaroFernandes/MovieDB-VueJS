Um pouco sobre o Projeto


Eu comecei o sem saber quase nada de JavaScript e muito menos sobre API, passei os primeiros dias apenas pesquisando sobre o assunto, eu ja sabia o basico de HTML e CSS então comecei fazendo a pagina inicial.


Inicialmente foi bem difícil entender como que funcionava a API pois eu achava que tinha que conectar como um banco de dados que eu já tinha feito antes e conectado com PHP, depois de analisar vários projetos eu entendi que era bem mais simples e que era tudo feito pelo link com a api key.


Nas minhas primeiras tentativas fui fazendo tudo do zero, porém como não tenho um conhecimento muito vasto em JavaScript e vue acabava que a sintaxe estava sempre errada. Eu consegui fazer os cards clicaveis utilizando as imagens como botões e assim abrindo um modal para passar as informções da API.
